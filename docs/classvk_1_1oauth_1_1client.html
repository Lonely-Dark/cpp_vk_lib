Detailed Description

VK Oauth client.

Note
Please be careful using this class in your open source code. Make sure that you don't store your personal data in string literals of your program. Much better to encapsulate that in an external config file.

Example usage:

int main() {
vk::oauth::client client(config::phone_number(), config::password(),
vk::oauth::target_client::windows);
client.pull();
vk::method::constructor(client.token())/ * ... * /.perform_request();
}
