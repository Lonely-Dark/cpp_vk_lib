# cpp_vk_lib
High performance VK API library

## Overview

* Convenient work with API methods
* Attachment handling system
* Built-in multithreading
* Logging
* OAuth-authorization support
* Buttons support

## Dependencies
	- curlpp

## Run tests

In `cpp_vk_lib` directory:
```
cmake .
cmake --build .
./build/test
```
