# cpp_vk_lib
Small and simple C++ library to facilitate work with Vkontakte API

## Dependencies
	- curlpp
	- simdjson

## Build shared object

In `cpp_vk_lib` directory:
```
cmake .
make
```

## Note
`examples` - a brief overview of the library's capabilities

`bot` - an example of a simple bot using `cpp_vk_lib`
